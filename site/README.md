# personal_design / site/design

This is a Vite + React portfolio site.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploy to Vercel

When importing this repository into Vercel:

- Framework Preset: `Vite`
- Root Directory: `site/design`
- Build Command: `npm run build`
- Output Directory: `dist`

If you bind a custom domain later, do it from the Vercel project settings after the first successful deployment.
